    Terminal : Creating new local file
  1. "mkdir GitPractice" > "cd GitPractice" > "ni index.html"> "start ./index.html"
  2. git init
  3. git add .
  4. git commit -m "take1"
  5. git remote add origin https://github.com/zulfidly/GitFeaturePractice.git
  6. git push -u origin main
  7. git remote -v (to check linkage)
    Terminal : Adding a file to GitHub
  1. "git add style.css"
  2. git commit -m "take2"
  3. git push origin main (style.css added to GitHub)
    Terminal : add & update all file to GitHub
  1. edit index.html & style.css
  2. git add .
  3. git commit -m "take3" (all files are now descripted "take3")
  4. git push origin main
    Terminal : add an item on GitHub, then pull it to local site
  1. README.md added on GitHub
  2. git pull origin main (README.md appeared on local site)
